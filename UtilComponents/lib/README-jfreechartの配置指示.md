# jfreechart-1.0.19.jar

This project uses JFreeChart to generate charts (graphs).

## Usage
The following source file depends on JFreeChart:
- UtilComponents/src/jfreechart/JFreeChartFunctions.java

## How to obtain
Download JFreeChart from the official website:
https://www.jfree.org/jfreechart/

## Placement
Place the jar file here:
UtilComponents/lib/jfreechart-1.0.19.jar

## Note
This jar is NOT included in this public repository.
Please download it separately and place it manually.
