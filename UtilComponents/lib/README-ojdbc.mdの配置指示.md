# ojdbc6.jar (Oracle JDBC Driver)

This project requires the Oracle JDBC driver to connect to an Oracle database.

## Usage
The following source files depend on this driver:
- UtilComponents/src/database/DBMS.java
- UtilComponents/src/utility/DbConnectUtil3.java

## License notice
The Oracle JDBC driver is subject to Oracle's license terms.
It is NOT redistributed in this public repository.

## How to obtain
Please download the appropriate version from Oracle:
https://www.oracle.com/database/technologies/appdev/jdbc-downloads.html

## Placement
Place the jar file here:
UtilComponents/lib/ojdbc6.jar
