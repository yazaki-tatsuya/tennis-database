# jcommon-1.0.23.jar

This library is a required dependency of JFreeChart.
It provides common utility classes used internally by JFreeChart.

## Usage
This project does not directly reference jcommon classes.
It is required at runtime by:
- jfreechart-1.0.19.jar

## How to obtain
jcommon is distributed together with JFreeChart.
Download it from:
https://www.jfree.org/jfreechart/

## Placement
Place the jar file here:
UtilComponents/lib/jcommon-1.0.23.jar

## Note
This jar is NOT included in this public repository.
It must be present together with jfreechart.jar to avoid runtime errors.
